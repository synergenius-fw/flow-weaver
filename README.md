# Flow Weaver

[![License: Elastic-2.0](https://img.shields.io/badge/License-ELv2-blue.svg)](./LICENSE)
[![CI](https://github.com/synergenius-fw/flow-weaver/actions/workflows/ci.yml/badge.svg)](https://github.com/synergenius-fw/flow-weaver/actions/workflows/ci.yml)

The TypeScript compiler for AI agent workflows.

Build visual workflows using JSDoc annotations and TypeScript function signatures. Full type safety, IDE autocomplete, and instant validation.
